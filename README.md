# React + Vite

cấu trúc dự án 

public: Chứa các tệp tĩnh như index.html.
src: Chứa tất cả mã nguồn React.
assets: Tài nguyên như hình ảnh và font chữ.
components: Các component tái sử dụng trong ứng dụng.
contexts: Các file sử dụng Context API để quản lý trạng thái ứng dụng.
hooks: Các custom hooks để tái sử dụng logic.
pages: Các thành phần trang (nếu ứng dụng có nhiều trang).
routes: Định nghĩa các route của ứng dụng.
services: Chứa các hàm gọi API và xử lý dữ liệu.
styles: Các tệp CSS hoặc SASS cho styling.
utils: Các hàm tiện ích dùng chung.
App.jsx: Component chính của ứng dụng.
main.jsx: Entry point, nơi khởi động ứng dụng.
